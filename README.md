### Emmet
Emmet是一款代码编辑器的插件, 支持多种流行的的编辑器. 通过它, 可以通过输入缩写来得到相应的HTML或者CSS代码, 大大提高输入效率.

[emmet-Syntax](http://docs.emmet.io/cheat-sheet/)

官方网站: <http://emmet.io/>
